word: 보이스 피싱 통화에 등장했던 단어

count: 기존에 보이스 피싱 통화에 등장했던 단어의 count 수를 가중치로 변환한 값


>> 보이스피싱 411건에 대해서 SST 진행 후 단어를 추출하고 횟수를 계산함.

>> excel파일 기준 상위에 있는 단어는 단어의 등장횟수가 많은 단어를 뜻함.

>> 단어의 등장 횟수를 0~1 MinMaxScaler을 진행하였음 ( 높은 단어일 수록 높은 가중치를 주기 위해 )

>> but 가중치가 0이면 연산이 안되기때문에 전부 1을 더해주어서 count의 range가 1~2가 되었음.

>> 임의로 가중치 합을 계산한 방법은  입력된 wav 파일에서 단어를 추출 후 word_weight에 해당하는 단어가 있으면 가중치를 더해주었음.
	ex)

	- 입력 텍스트 : OO캐피탈에 은행 계좌를 증설하셨죠?
	- 단어 추출 : [ OO캐피탈, 은행, 계좌, 증설 ]

 	- 가중치 계산
		캐피탈 : 1.08
		은행 : 1.98
		계좌 : 1.74
		증설 : 단어 리스트에 없음
		입력 텍스트에 대한 임의로 정한 가중치 값 : 1.08 + 1.98 + 1.74 = 4.8
